## Using
Download the script 
```bash
git clone https://github.com/Rus121/ftp-server.git
```
Open the folder
```bash
cd ftp-server
```
Right permission for the file
```bash
chmod +x ftp.sh
```
Execute script
```bash
sudo ./ftp.sh
```
